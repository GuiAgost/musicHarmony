Como transpor uma nota ou um acorde?

    Para Transpor uma nota ou um acorde, é alterar para uma região mais grave ou mais aguda (aumentar ou diminuir). Vamos supor uma música tem o tom de acorde C, então queremos diminuir 3 semitons. Utilizando o relógio para calcular a distância, vamos para o sentido anti-horário a partir do acorde C e assim encontramos o acorde resultado A:

    Ou seja, a distância entre o acorde C para o acorde A é de menos 1 tom e 1 semitom.

    Agora, queremos aumentar do acorde G em 5 semitons. Utilizando mais uma vez o relógio para calcular, vamos para o sentido horário a partir do acorde G e assim encontramos o acorde resultado C:

    Sendo assim, a distância entre o acorde G para o acorde C é mais de 2 tons e 1 semitom. Será realizada a mesma lógica na sequência de acordes de uma música e também para as notas de um solo musical.
